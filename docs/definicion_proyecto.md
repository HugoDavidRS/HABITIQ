# Definición del Proyecto HabitIQ

## Objetivo
Crear una aplicación web pública para gestión de hábitos que demuestre habilidades full-stack.

## Público Objetivo
- Personas que buscan mejorar hábitos diarios
- Desarrolladores que buscan ejemplos de portafolio
- Estudiantes de programación

## Características Técnicas
- Arquitectura limpia y modular
- Código documentado y testeable
- Preparado para escalar
- Orientado a buenas prácticas

## Valor Diferencial
1. **Simplicidad:** Fácil de usar
2. **Extensible:** Fácil de modificar
3. **Educativo:** Código bien estructurado
4. **Profesional:** Listo para portafolio
